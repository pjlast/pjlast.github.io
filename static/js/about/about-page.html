About this app

This "app" uses a make-shift client-side router, which you can find here. 

const path = window.location.pathname;
if (path === "/") {
  // handle index route
  import("./index/component.js");
} else {
  import(`.${path}/component.js`);
}

By reading the URL path, we can dynamically import a web component that implements page-content, which is then inserted into the HTML template. In this GitHub Pages app, each page will still have its own .html file, because that is just how Git Hub Pages works. However, in a server where you would traditionally do templating, you can serve the same page for all routes.
